Java Import 説明

インポートされたクラスやパッケージの説明
Import 意味 用途
java.awt.BorderLayout ウィンドウの領域（北・南・東・西・中央）にコンポーネントを配置するレイアウトマネージャー GUIのレイアウトを整える
java.awt.Color 色を指定するためのクラス 背景色や文字色の変更
java.awt.Font フォントのスタイルやサイズを設定するクラス テキストのデザイン変更
java.awt.GridLayout 格子状にコンポーネントを配置するレイアウトマネージャー 均等な配置
java.time.LocalDate 日付を扱うためのクラス 今日の日付や特定の日付を取得・操作
java.time.YearMonth 年と月を扱うクラス カレンダー表示や月末日取得
java.util.ArrayList 動的配列を提供するクラス リスト管理
java.util.HashMap キーと値のペアを管理するマップ構造 データ検索や関連付け
javax.swing.JButton ボタンを表すクラス クリック可能なボタンの作成
javax.swing.JDialog ダイアログウィンドウを作成するクラス ポップアップウィンドウの作成
javax.swing.JFrame ウィンドウを作成するクラス メインウィンドウの作成
javax.swing.JLabel ラベルを作成するクラス テキストや画像の表示
javax.swing.JPanel コンポーネントをグループ化するコンテナクラス コンポーネントのまとめと配置
javax.swing.JScrollPane スクロール可能なビューを提供するクラス スクロールバーの追加
javax.swing.JTextArea 複数行のテキスト入力・表示ができるコンポーネント 長いテキストの表示や入力
javax.swing.SwingConstants 位置指定用の定数を提供するインターフェース コンポーネントやテキストの配置
javax.swing.SwingUtilities Swingのスレッド関連ユーティリティクラス GUI更新を安全に行う
